这篇文章展示了 Su 主题支持的各种 Markdown 语法。

## 标题

使用 `#` 符号创建标题，从 H1 到 H6。

## 文本格式

**粗体文本** 使用 `**粗体**`

*斜体文本* 使用 `*斜体*`

~~删除线~~ 使用 `~~删除线~~`

## 链接

[访问 Hugo 官网](https://gohugo.io)

## 图片

虽然 Su 主题是纯文字主题，但仍然支持图片：

```markdown
![Alt text](image-url.jpg)
```

## 代码块

行内代码：`const x = 42;`

代码块：

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Su!")
}
```

## 引用

> 这是一个引用块。
>
> 可以包含多个段落。

## 列表

### 无序列表

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
- 项目 3

### 有序列表

1. 第一步
2. 第二步
3. 第三步

## 表格

虽然不常用，但 Markdown 也支持表格：

| 特性 | 支持 |
|------|------|
| 极简设计 | ✓ |
| 深色主题 | ✓ |
| 响应式 | ✓ |

---

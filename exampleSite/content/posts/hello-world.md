欢迎使用 **Su (素)** 主题！

## 关于 Su

Su 是一个极简主义的 Hugo 主题，专注于提供纯粹的阅读体验。

### 主要特性

- **极简设计** - 没有多余的装饰，让内容成为焦点
- **深色主题** - 护眼舒适的深色配色
- **响应式布局** - 在任何设备上都有良好表现
- **代码高亮** - 支持语法高亮和一键复制

### 代码示例

这是一个 Python 代码示例：

```python
def hello_world():
    print("Hello, Su Theme!")
    return "Welcome to minimalism"

if __name__ == "__main__":
    hello_world()
```

这是一个 JavaScript 示例：

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
    return `Welcome to Su Theme`;
}

greet("World");
```
